# 🐍 Snake Game (Python Turtle)

A classic **Snake Game** built using Python's built-in **turtle graphics** module.
Control the snake, eat food, grow longer, and avoid colliding with the wall or your own tail!

---

## 🎮 Features

* Smooth snake movement using **arrow keys**
* **Random food spawning**
* **Live score tracking**
* Snake **grows longer** after eating food
* **Game Over conditions**

  * Snake hits the wall
  * Snake hits its own tail
* Modular code using **Python classes**

---

## 📂 Project Structure

```
snake_game/
│
├── main.py        # Runs the game loop
├── snake.py       # Snake creation, movement, and growth
├── food.py        # Food spawning and refresh logic
├── scoreboard.py  # Score display and game over message
```

---

## ▶️ How to Run

1. Clone the repository

```
git clone https://github.com/your-username/snake-game.git
```

2. Navigate to the project folder

```
cd snake-game
```

3. Run the game

```
python main.py
```
